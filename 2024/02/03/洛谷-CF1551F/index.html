Equidistant Vertices

题意

给你一个nnn个点的树，要求选出kkk个点，使得这kkk个点两两距离相同，问你这样的kkk个点的方案有多少种，答案取模1e9+71e9 + 71e9+7，一共ttt组

数据范围

2≤k≤n≤1002 \leq k \leq n \leq 1002≤k≤n≤100

1≤t≤101 \leq t \leq 101≤t≤10

思路

通过构造不难发现，当k==2k == 2k==2的时候时候答案就是 (n2)\tbinom{n}{2}(2n​)，

当k>2k > 2k>2的时候，一定存在一个分叉点，这kkk个点达到分叉点的距离都相同（我也不知道怎么证明，

那么这样之后，我们其实就可以求出每个点为根（分叉点）的时候，每个子树（包括父亲这个子树）中，距离分叉点为d,1≤d<nd, 1 \leq d < nd,1≤d<n的数量，现在我们枚举了距离ddd，还有枚举了根uuu，我们就可以得到一个序列，其中第iii个点表示uuu的第iii个子树中，距离uuu的长度为ddd的节点数量（为000的就不用管了），假设大于数量大于000的有mmm个，

那么其实就是这个mmm个里面取kkk个，当时想着用组合数来做（但是一直没想到解法），看了题解后知道可以用dpdpdp，那么其实dp[i][j]dp[i][j]dp[i][j]就表示前iii个数中取了kkk个数的方案数，那么转移方程就是

dp[i][j]=dp[i−1][j]+dp[i−1][j−1]∗aidp[i][j] = dp[i - 1][j] + dp[i - 1][j - 1] * a_idp[i][j]=dp[i−1][j]+dp[i−1][j−1]∗ai​

那么ansansans就加等于dp[m][k]dp[m][k]dp[m][k].
