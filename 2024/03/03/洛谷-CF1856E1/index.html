PermuTree (easy version)

题意

这个是问题的简单版本，两个版本的区别就是nnn的范围不同，时间限制不同

给定一棵111为根的树，然后那你用一个[1,n][1, n][1,n]的全排列aaa，来给树上每个节点赋值，使得满足条件的二元组(u,v)(u, v)(u,v)数量最多，

条件：

au<alca(u,v)<ava_u < a_{lca(u, v)} < a_vau​<alca(u,v)​<av​

数据范围

2≤n≤50002 \leq n \leq 50002≤n≤5000

思路

我的想法，就是每个子树里面都是一段连续的序列，但是我不清楚怎么证明！

假设我上面的成立的话，

那么其实对于一棵子树，就是确定根所选的点xxx，那么这个子树的贡献就是[1,x−1]×[x+1,子树大小−1][1, x - 1] \times [x + 1, 子树大小 - 1][1,x−1]×[x+1,子树大小−1]，

我们知道这个子树的大小是恒定的，那么其实就是求x−1x - 1x−1和子树大小−1−(x+1)+1子树大小-1-(x+1)+1子树大小−1−(x+1)+1乘积的最大值，

这个其实就是求两部分最接近的时候，也就是最逼近子树大小的一半，这个就可以用背包来解决
