P2967 [USACO09DEC] Video Game Troubles G

题意

给你nnn个游戏机，每个游戏机有专属游戏，玩这些游戏可以获得牛奶产量pvjpv_jpvj​，但是你要玩这个游戏，要先买对应的游戏机pip_ipi​，然后再买这个游戏GPjGP_jGPj​，现在假设你有VVV元，问你怎么样牛奶的产量最大

数据范围

1≤n≤501 \leq n \leq 501≤n≤50

1≤pi≤10001 \leq p_i \leq 10001≤pi​≤1000

1≤GPj≤1001 \leq GP_j \leq 1001≤GPj​≤100

1≤pvj≤10000001 \leq pv_j \leq 10000001≤pvj​≤1000000

1≤V≤1000001 \leq V \leq 1000001≤V≤100000

思路

一开始想得是依赖背包，但是没有优化的空间，只能想到O(nV2)O(nV^2)O(nV2)的做法，GG

后面看了题解，我们可以考虑每个游戏机是不是要买，然后买的情况下，就是正常的01背包，然后再考虑不买的情况，这样就得到最终的答案。

注意点

你01背包的时候，容量至少要大于等于游戏机 + 对应游戏的价格。
