Salazar Slytherin’s Locket

题意

给你区间[l,r][l, r][l,r]，qqq次询问，让你求出区间中有多少个数字满足下列的性质，

  • 给定一个进制bbb，这个数字转换为bbb进制后，0,1,⋯ ,b−10, 1, \cdots, b - 10,1,⋯,b−1都出现偶数次数

数据范围

1≤q≤1051 \leq q \leq 10^51≤q≤105

1≤b≤101 \leq b \leq 101≤b≤10

1≤l≤r≤10181 \leq l \leq r \leq 10^{18}1≤l≤r≤1018

思路

我们首先要知道怎么把十进制转换为其它进制，

比如32转换为3进制：

  • 32除以3：商为10，余数为2
  • 10除以3：商为3，余数为1
  • 3除以3：商为1，余数为0
  • 1除以3：商为0，余数为1

那么32对应的三进制就是1012

接着不然发现这个是个数位DP的题目，我们就是计算sum[r]−sum[l−1]sum[r] - sum[l - 1]sum[r]−sum[l−1]，其中sum[n]sum[n]sum[n]表示[1,n][1, n][1,n]中满足题意的数字的数量，

我们可以求出所有小于nnn的方案，然后再判断nnn是不是满足即可

小于nnn的数字可以分成iii类，其中第iii类表示第iii位开始小于nnn的数字，比如n=12345n = 12345n=12345，那么一共就有m=5m = 5m=5类

  • 第1类：[1, 9999]
  • 第2类：[10000, 11999]
  • 第3类：[12000, 12299]
  • 第4类：[12300, 12399]
  • 第5类：[12340, 12344]

那么每一类中，我们怎么快速求出方案数呢？

假设我们目前考虑第iii类，那么我们就先枚举第iii位的数字，
那么[i+1,m][i + 1, m][i+1,m]的每一位都可以是[0,b−1][0, b - 1][0,b−1]，我们设没一个数字的奇偶状态为0/1，那么[0,b−1][0, b - 1][0,b−1]的奇偶状态就可以是一个二进制，
那么我们假设已知前[1,i][1, i][1,i]的奇偶状态为010010100101001，那么是不是后面[i+1,m][i + 1, m][i+1,m]的状态也要为010010100101001，

那么也就是说我们要知道在bbb进制状态下，前iii位状态为jjj的方案数，那么这个可以用DP预处理下就好.
