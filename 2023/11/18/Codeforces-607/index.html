B - Zuma

题意

给你一个长度为nnn的数组，第iii位的值是aia_iai​，你现在可以每次选择一个子数组，这个子数组是回文数组，然后花费111的代价把这个子数组去掉

问你把整个数组去掉的最小代价是多少

数据范围

1≤n≤5001 \leq n \leq 5001≤n≤500

1≤ai≤n1 \leq a_i \leq n1≤ai​≤n

思路

设方程为f[i][j]f[i][j]f[i][j]表示去除[i,j][i, j][i,j]的最小代价，

不然发现，如果ai==aja_i == a_jai​==aj​的话，f[i][j]f[i][j]f[i][j]可以从f[i+1][j−1]f[i + 1][j - 1]f[i+1][j−1]直接转移过来，不需要花费额外的代价，因为每次消除的时候一定是一个回文串，那么回文串首尾都加一个相同的数字，肯定也是一个回文串，那么就可以一起消除

如果ai≠aja_i \neq a_jai​=aj​的话，我们就枚举分界点kkk，然后f[i][j]=min(f[i][j],f[i][k]+f[k+1][j]f[i][j] = min(f[i][j], f[i][k] + f[k + 1][j]f[i][j]=min(f[i][j],f[i][k]+f[k+1][j]，就相当于是分成两段独立消除
