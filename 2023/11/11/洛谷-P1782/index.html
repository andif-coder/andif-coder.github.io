P1782 旅行商的背包

题意

有nnn种物品，每种物品的体积为ViV_iVi​，价值为WiW_iWi​，一共有DiD_iDi​件

还有一批物品，有mmm种，第iii件的价值YiY_iYi​和体积XiX_iXi​的关系如下，Yi=aiXi2+biXi+ciY_i = a_iX_i^2 + b_iX_i + c_iYi​=ai​Xi2​+bi​Xi​+ci​。

问你背包体积为CCC的时候，最大的价值为多少。

数据范围

1≤n≤1041 \leq n \leq 10^41≤n≤104

1≤m≤51 \leq m \leq 51≤m≤5

1≤C≤1041 \leq C \leq 10^41≤C≤104

1≤Wi,Vi,Di≤10001 \leq W_i, V_i, D_i \leq 10001≤Wi​,Vi​,Di​≤1000

−1000≤ai,bi,ci≤1000-1000 \leq a_i,b_i,c_i \leq 1000−1000≤ai​,bi​,ci​≤1000

思路
