P1853 投资的最大效益

题意

给你一些债券，每种债券至少投资aia_iai​，一年后可以收到ai+bia_i + b_iai​+bi​的钱，

已知你的初始资产为sss，有ddd种债券，问你怎么投资，可以在nnn年后，获得最大的总资产

数据范围

1≤s≤1061 \leq s \leq 10^61≤s≤106

2≤n≤402 \leq n \leq 402≤n≤40

1≤d≤101 \leq d \leq 101≤d≤10

1≤a≤1041 \leq a \leq 10^41≤a≤104 并且 aaa一定是100010001000的倍数

bbb不超过aaa的10%10\%10%

思路

这样我们每年都做个完全背包的话，显然复杂度不够，

我们观察数据范围，发现aaa一定是100010001000的倍数，那么其实，每一年初始资产s%1000s \% 1000s%1000的部分是没有用的，
