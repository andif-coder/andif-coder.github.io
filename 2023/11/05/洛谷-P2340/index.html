P2340 [USACO03FALL] Cow Exhibition G

题意

有nnn只奶牛，第iii只奶牛的智商为sis_isi​，情商为fif_ifi​，

现在要你选择几只（可以不选），问你在满足下列两种条件的情况下，

  • 奶牛情商之和大于等于零
  • 奶牛智商之和大于等于零

情商和智商的和的最大值为多少

数据范围

1≤n≤4001 \leq n \leq 4001≤n≤400

−1000≤si,fi≤1000-1000 \leq s_i, f_i \leq 1000−1000≤si​,fi​≤1000

思路

我们可以固定一维，我们设状态方程为dp[i][j]dp[i][j]dp[i][j]，表示前iii个奶牛，情商恰好为jjj的情况下，最大的智商和。

但是因为有负数的情况，所以我们需要把jjj统一加上m=1000∗nm = 1000 * nm=1000∗n，这样保证不会溢出。

由于二维的内存会炸掉，我们可以用类似01背包内存优化的方法，来优化内存

只是由于f[i]f[i]f[i]有正有负，在大于等于000的情况下，我们要逆序枚举jjj，在小于0的情况下，我们要正序枚举jjj

注意点

在枚举的时候，jjj要同时满足下面两个条件：

  • 0≤j≤2∗m0 \leq j \leq 2 * m0≤j≤2∗m
  • 0≤j−f[i]≤2∗m0 \leq j - f[i] \leq 2 * m0≤j−f[i]≤2∗m

当f[i]≥0f[i] \geq 0f[i]≥0的时候，上面的范围的交集是：

  • f[i]≤j≤2∗mf[i] \leq j \leq 2 * mf[i]≤j≤2∗m

当f[i]<0f[i] < 0f[i]<0的时候，上面范围的交集是：
