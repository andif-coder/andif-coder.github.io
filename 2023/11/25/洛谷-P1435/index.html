P1435 [IOI2000] 回文字串

题意

给你一个长度为nnn的字符串，然后问你最少填充几个字符，可以让这个字符串变成回文串

数据范围

1≤n≤10001 \leq n \leq 10001≤n≤1000

思路

设状态为dp[i][j]dp[i][j]dp[i][j]表示区间[i,j][i, j][i,j]的子串变成回文串的最少需要字符数量，

如果s[i]==s[j]s[i] == s[j]s[i]==s[j]，那么dp[i][j]=dp[i+1][j−1]dp[i][j] = dp[i + 1][j - 1]dp[i][j]=dp[i+1][j−1]，为什么这个一定是最优呢，我们可以假设一边s[i]s[i]s[i]去匹配里面的字符串，然后另外一个s[j]s[j]s[j]就得需要新增一个才能字符才能被匹配，而s[i]s[i]s[i]又最多减少一个内部字符串需要的字符，显然是不赚的。

如果s[i]≠s[j]s[i] \neq s[j]s[i]​=s[j]的话，那么就有两种情况，我们可以用一个新的去匹配s[i]s[i]s[i]，那么dp[i+1][j]+1dp[i + 1][j] + 1dp[i+1][j]+1，或者就是用一个新的去匹配s[j]s[j]s[j]，那么就是dp[i][j−1]+1dp[i][j - 1] + 1dp[i][j−1]+1.
